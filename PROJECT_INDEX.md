## Hook System Flow

```
SessionStart hook (caveman-activate.js)
  → writes mode to $CLAUDE_CONFIG_DIR/.caveman-active via safeWriteFlag()
  → emits ruleset as hidden stdout (system context injection)

UserPromptSubmit hook (caveman-mode-tracker.js)
  → /caveman [lite|full|ultra|wenyan*] → writes mode to flag
  → "stop caveman" / "normal mode" → deletes flag
  → per-turn reinforcement when flag is set

caveman-statusline.sh
  → reads flag → outputs [CAVEMAN] or [CAVEMAN:<MODE>] badge
```

Flag file: `$CLAUDE_CONFIG_DIR/.caveman-active` (fallback: `~/.claude/.caveman-active`)

**Security:** All flag writes go through `safeWriteFlag()` — refuses symlinks, atomic temp+rename, `0600` perms.

## Eval Architecture

3-arm harness — honest delta = **skill vs terse** (not skill vs baseline):
- `__baseline__` — no system prompt
- `__terse__` — `"Answer concisely."`
- `<skill>` — `"Answer concisely.\n\n{SKILL.md}"`

Snapshots committed to git. CI reads without API calls. Only regenerate when SKILL.md or prompts change.
